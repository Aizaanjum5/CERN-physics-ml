# CERN Physics Machine Learning Project

This project applies machine learning to CMS Open Data to predict particle mass and extract insights using regression models.

## Project Overview

- **Goal:** Predict the mass (`M`) of particles using features from the CERN CMS dataset.
- **Dataset:** CMS Open Data (processed and feature engineered for regression tasks)
- **Tools:** Python, pandas, scikit-learn, matplotlib, joblib
